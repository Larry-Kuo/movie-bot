# TOC Project 2021 - Movie-bot

## 簡介
	1.可以查詢想看的電影，幫忙在ptt上面用爬蟲檢查負雷/好雷的出現次數，確保想看的電影不會太難看
	2.若是不知道想看什麼電影，點選電影推薦可以根據電影類型推薦IMDB前250名的電影
	3.透過IMDB查詢特定電影（前250）的基本資訊（片長、導演、年份...等）
